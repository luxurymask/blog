踩坑：[python] selenium报错

老师让改个爬虫支持爬取网页由js动态加载的部分，需要用selenium来做（好像比较方便，比webkit方便？）。

以前电脑上安装过python 2.7，直接装下selenium就行。

pip install selenium不太好使，不知道咋回事儿。就换了easy_install selenium来安装。

Searching for selenium

Best match: selenium 3.4.2

Adding selenium 3.4.2 to easy-install.pth file

Using /Library/Python/2.7/site-packages

Processing dependencies for selenium

Finished processing dependencies for selenium

应该是成功了。

然后试一下，看能不能调到firefox，结果出了问题。

>>> from selenium import webdriver

>>> driver = webdriver.Firefox()

Traceback (most recent call last):

File "", line 1, in

File "/Library/Python/2.7/site-packages/selenium/webdriver/firefox/webdriver.py", line 142, in __init__

self.service.start()

File "/Library/Python/2.7/site-packages/selenium/webdriver/common/service.py", line 81, in start

os.path.basename(self.path), self.start_error_message)

selenium.common.exceptions.WebDriverException: Message: 'geckodriver' executable needs to be in PATH.

>>> exit()

咋回事儿，查了一下，是firefox驱动'geckodriver'没加好。

在http://selenium-python.readthedocs.io/下载了驱动，解压到/usr/local/bin，好了。

>>> from selenium import webdriver

>>> driver = webdriver.Firefox()

>>>

firefox成功打开，收工。
